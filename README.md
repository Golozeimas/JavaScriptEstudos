# Estudos de JavaScript

Repositório com exercícios e exemplos práticos desenvolvidos durante os estudos de JavaScript.

---

## Estrutura do Projeto

| Pasta | Descrição |
|---|---|
| `Variaveis e manipulação` | Introdução a variáveis, `document.write` e loops básicos |
| `Var_let_e_const` | Diferenças entre `var`, `let` e `const`, escopo e imutabilidade |
| `Condicionais e comparação` | Operadores de comparação (`==` vs `===`) e operador ternário |
| `loops` | Estruturas de repetição: `while` e `for` |
| `switchs` | Uso do `switch/case` com interação via `prompt` |
| `Funções` | Funções, manipulação do DOM e eventos de clique |
| `FunçõesPt2` | Funções com parâmetros, retorno e lógica condicional |
| `FunçõesAnônimas` | Arrow functions e uso do método `map` |
| `Templates Strings` | Template literals e interpolação de variáveis |
| `Arrays` | Introdução a arrays *(em desenvolvimento)* |
| `OperaçõesComArrays_Map_Reduce_Filter` | Métodos `map`, `reduce`, `filter` e `find` |
| `Includes_StartsWith_E_EndsWith` | Métodos de busca em strings e arrays |
| `Find` | Uso do `find` em arrays simples e arrays de objetos |
| `Objetos` | Criação, acesso, alteração e iteração de objetos |
| `Desconstruindo_Arrays_E_Objetos` | Destructuring de arrays e objetos |
| `SpreadOperator` | Operador spread (`...`) em arrays e objetos |
| `RestOperator` | Operador rest em funções com número variável de argumentos |
| `Datas` | Objeto `Date`, formatação e manipulação de datas |
| `Temporizadores` | `setInterval` e `clearInterval` |
| `RequisiçõesHTTP` | Consumo de API com `fetch` e Promises |
| `CalculadoraIMC` | Projeto prático: calculadora de IMC com HTML, CSS e JS |
| `Desafios` | Exercícios de fixação sobre listas, objetos, funções e datas |

---

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```
2. Abra a pasta do tópico desejado no VS Code.
3. Use a extensão **Live Server** (porta `5501`) para visualizar os exemplos no navegador.
